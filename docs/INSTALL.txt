tx.tinymce_linguaplone_patch Installation
-----------------------------------------

If you are using buildout you can do this:

* Add ``tx.tinymce_linguaplone_patch`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        tx.tinymce_linguaplone_patch
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout -N
